# wordguess
aka
## UFO: The Game  

This application is a word guess game, similar to hangman, run in the command line using Node.js  

**INSTRUCTIONS**  

* open your command-line terminal  
* `npm install`  

**TO PLAY**  
`npm run start`  

**TO EXIT** ***mac users***    
`control + c`

**RUN TESTS**  
`npm run test`
